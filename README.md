# java-dsa-practice

Welcome to my Java Basics practice repository!  

This repo contains simple Java programs focusing on fundamental programming concepts that every beginner should know.

---

## Topics Covered

- If-Else Statements  
- For Loops  
- While Loops  
- Pattern Printing

---

## Program Files

- `IfElseExample.java` — Demonstrates conditional statements with if-else.  
- `ForLoopExample.java` — Shows how to use for loops for repetition.  
- `WhileLoopExample.java` — Uses while loops to repeat actions.  
- `PatternExample.java` — Prints a simple star pattern using nested loops.

---

## How to Run

1. Clone the repository  
2. Open the `.java` files in your preferred Java IDE (e.g., IntelliJ, Eclipse, VS Code)  
3. Compile and run the files using the IDE or command line
